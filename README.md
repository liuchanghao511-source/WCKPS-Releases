## Installation

1. Download `WCKPS_Pro.zip` from the latest release
2. Extract to desired location
3. Run `WCKPS_Pro.exe`
4. On first launch, enter your Gemini API key in System Configuration

## Requirements

- Windows 10/11
- Microsoft Outlook (for email sync)
- Network access to store NAS paths

## Auto-Update

WCKPS Pro checks this repository for new versions on startup.  
Updates are downloaded and applied automatically — no manual reinstall needed.

For issues or feedback, use the in-app Feedback button or contact your system administrator.
